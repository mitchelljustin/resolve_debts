# Resolve Debts
Be the last one out to get this dough? No way

## Installation
````bash
pip3 install -r requirements.txt
````

## Usage

#### Generate Ledger

````bash
python3 generate_ledger.py > someledger.yml
````

#### Resolve Debts

````bash
python3 resolve_debts.py -f someledger.yml --draw
````
